# 🚀 GitControl - GitHub Management Tool for Termux

A comprehensive command-line tool for managing your GitHub account and repositories directly from Termux.

## ✨ Features

- 📋 List all your repositories
- 📁 Create new repositories
- 🗑️ Delete repositories
- 📊 View repository statistics
- 🔍 Search repositories
- 📝 View user profile
- ⭐ Star repositories
- 🔀 Fork repositories
- 📦 List and create issues
- 📈 View repository activity
- 🔑 Manage collaborators
- 📤 Upload files to repositories
- 📥 Download files from repositories
- 🔄 Refresh GitHub token

## 📱 Installation

### Automatic Installation
```bash
curl -o gitcontrol_setup.sh https://raw.githubusercontent.com/yourusername/gitcontrol/main/setup.sh
